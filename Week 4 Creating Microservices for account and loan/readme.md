## Overview

This exercise demonstrates the development of two independent Spring Boot REST microservices:
- Account Microservice
- Loan Microservice

## Features

- Developed using Spring Boot
- RESTful API implementation
- Separate Account and Loan services
- JSON response format
- Tested using Browser/Postman
- Maven project structure

## API Endpoints

### Account Service
```
GET /accounts/{number}
```

### Loan Service
```
GET /loans/{number}
```

## Technologies Used

- Java
- Spring Boot
- Maven
- REST API
- JSON
